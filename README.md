# Sales Analysis Dashboard (Power BI)

## Project Overview
This project analyzes retail sales data to identify key business insights and performance trends using Power BI.

## Tools Used
- Power BI
- Excel / CSV

## Key KPIs
- Total Revenue: $3.13M
- Average Order Value: $1.25K
- Total Orders: 2,502
- Total Quantity Sold: 63K

## Dashboard Features
- Monthly Revenue Trend Analysis
- Top Products by Revenue
- Revenue by Country
- Year-wise filtering
- Business insights and recommendations

## Key Insights
- Revenue shows a continuous decline throughout the year.
- A few products contribute a major share of total revenue.
- France and the United States are the top-performing markets.

## Business Recommendations
- Investigate reasons for declining demand.
- Run targeted promotions in low-performing months.
- Focus marketing on high-performing regions.
- Optimize product mix.
